Design principles

For this task, I utilised the following 3 design principles: Contract, Repetition & Interactivity.

Contrast

For the design principle of contrast, I implemented it by using contrasting colours for the backgrounds of my website, as well as for the text. The primary colours I used were “Facebook Blue”, colour code: #3b5998, and plain white. I also used silver as the intermediary text colour.

Repetition

For the design principle of repetition, I decided to utilise the same navigation layout (global nav bar with clickable links to each page) on each page, allowing them to be accessed from all areas.

Interactivity

Finally, for the design principle of Interactivity, I made sure that the global navigation bar had the appropriate section highlighted and clickable so that the user knew where they were located.
